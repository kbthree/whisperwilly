FAQ

How do I change the hero video?

Use Set Background (top right) to pick a local clip while developing. For production, drop .mp4 files in /assets/video and update the clips array in the script.

Where do I put my repo links?

Edit the clips array: each item has a href that points to a GitHub folder (e.g., tree/main/examples).

How do I enable the forum?

Enable Discussions in your GitHub repo, create a category (e.g., Q&A), then paste your data-repo, data-repo-id, data-category-id into the giscus script above.

Will this support Google/Confluence sign‑in later?

Yes. The UI already segments features. We’ll add GIS/PKCE flows and wire the buttons to their APIs.
